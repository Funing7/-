##### 5.图片

```
<img src="" alt="" title="" width="">
各个属性间用空格隔开
src     放图片的地址
alt     如果地址失效显示的文字
title   悬停时显示的文字
width   图片的宽
```

##### 6.路径

```
1.绝对路径
(1)直接从网络上复制的图片的链接
(2)从盘符开始一套一套直到图片
2.相对路径
(1)同级
	1.直接写名字 src="图片.png"	
	2.src="./图片.png"
(2)下级
	src="source/图片.png"
(3)上级
	src="../source/东秦鸡鸣湖.png"
```
